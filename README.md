# Slither

Juego de Snake desarrollado con HTML, CSS y JavaScript.

## Instalación

1. Clona el repositorio:
```bash
git clone https://github.com/CarlosDaniel-GCH/slither.git
```

2. Abre `index.html` en tu navegador.

## Controles

- **Flechas direccionales**: Mover la serpiente
- **Espacia**: Pausa/Reanudar

## Características

- Movimiento fluido de la serpiente
- Sistema de puntuación
- Colisiones con paredes y cuerpo
- Diseño responsivo
